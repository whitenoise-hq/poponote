# 디자인 참고본 (Figma Make)

Figma Make(Pet Diary Mobile App)로 만든 화면 캡처. 똑같이 구현하는 게 아니라
레이아웃·무드 참고용이며, 이모지·텍스트·색은 우리 토큰(코랄/크림 + Gothic A1/Jua)에 맞춰 조정한다.

## 캡처

- `home.png` — 홈 탭
- `diary1.png` / `diary2.png` — 다이어리 탭 (캘린더 / 피드)
- `album.png` — 일러스트 앨범 탭
- `settings1.png` / `settings2.png` — 설정 탭

## 컬러 토큰

- `figma-tokens.md` — Figma 원본 `:root` 색상값 + 우리 Tailwind 토큰 매핑
